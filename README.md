# 肉じゃがシステム

## 公開用ファイル

- `index.html`：サイト本体
- `guide.pdf`：冊子「志願理由書の書き方Lv999」
- `resume.pdf`：2ページ構造レジュメ
- `favicon.ico`：ブラウザー用favicon
- `favicon.png`：高解像度アイコン／スマートフォン用アイコン
- `pot-icon.svg`：鍋アイコンのSVG原本
- `.nojekyll`：GitHub Pages用

## GitHub Pagesへの配置

このフォルダの中身を、公開するリポジトリのルートへそのままアップロードしてください。

GitHubのリポジトリで、`Settings` → `Pages` を開き、公開元を `Deploy from a branch`、ブランチを `main`、フォルダを `/(root)` に設定します。

## 注意

- `index.html`、`guide.pdf`、`resume.pdf` のファイル名は変更しないでください。サイト内のリンクで使用しています。
- Google Fontsはオンラインから読み込むため、オフライン環境では代替フォントで表示されます。
- パスワード認証は、URLの不用意な共有を抑えるための簡易ロックです。

- `images/nikujaga-method.jpg`：肉じゃが法の説明画像（通常表示・拡大表示用）
